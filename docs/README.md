# Docs

Pasta para documentos auxiliares, prompts e materiais de apoio ao aprendizado.

## Arquivos

| Arquivo | Descricao |
| --- | --- |
| `python-mentor.txt` | Prompt para orientar respostas de mentoria em Python, com foco em aprendizado gradual e boas praticas. |

## Uso

Use esta pasta para guardar materiais que ajudam a estudar, revisar ou padronizar fluxos de trabalho, mas que nao pertencem diretamente a um curso ou projeto especifico.
