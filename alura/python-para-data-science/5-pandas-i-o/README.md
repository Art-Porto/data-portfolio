# 5 - Pandas I/O

Modulo voltado para entrada e saida de dados com Pandas em diferentes formatos.

## Arquivos

| Arquivo | Descricao |
| --- | --- |
| `Aula.ipynb` | Notebook principal do modulo. |
| `Desafio.ipynb` | Desafios de leitura, tratamento e exportacao. |
| `clientes_mercado.csv` | Base de clientes. |
| `dados_mercado.csv` | Base de mercado. |
| `dados_sus.csv` | Base de dados publica para pratica. |
| `Emissoes_CO2.csv` | Base de emissoes de CO2. |
| `co2_percapita.xlsx` | Base em Excel. |
| `historico_pacientes_norm.json` | Base em JSON. |
| `filmes_imdb.xml` | Base em XML. |
| `top_filmes.html` | Tabela em HTML para leitura com Pandas. |
| `top_filmes_1998` | Arquivo de apoio relacionado ao exercicio de filmes. |

## Temas estudados

- Leitura de CSV, XLSX, JSON, XML e HTML.
- Inspecao de fontes diferentes.
- Conversao e exportacao de dados.
- Organizacao de bases para analise.
