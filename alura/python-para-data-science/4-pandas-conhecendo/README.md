# 4 - Pandas: conhecendo a biblioteca

Modulo introdutorio de Pandas para leitura, exploracao e tratamento inicial de dados tabulares.

## Arquivos

| Arquivo | Descricao |
| --- | --- |
| `notebook_inicial.ipynb` | Notebook principal com conceitos de Pandas. |
| `desafios.ipynb` | Desafios praticos do modulo. |
| `dados_apartamentos.csv` | Base de apartamentos. |
| `dados_completos_dev.csv` | Base de dados completa para pratica. |
| `alunos_aprovados.csv` | Base derivada ou tratada nos exercicios. |

## Temas estudados

- DataFrames e Series.
- Leitura de arquivos CSV.
- Inspecao de dados.
- Filtros e selecoes.
- Tratamento inicial e criacao de novas bases.
