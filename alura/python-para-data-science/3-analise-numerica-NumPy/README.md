# 3 - Analise numerica com NumPy

Modulo dedicado a calculos numericos com NumPy e comparacoes de desempenho.

## Arquivos

| Arquivo | Descricao |
| --- | --- |
| `Aula_Curso_Numpy_Alura.ipynb` | Notebook principal do curso de NumPy. |
| `Praticando.ipynb` | Exercicios e praticas complementares. |
| `Test-time-array.ipynb` | Testes de tempo e comparacao de operacoes com arrays. |
| `Dados.csv` | Base de dados usada nas praticas. |
| `raw.githubusercontent.com/` | Copias locais de bases baixadas de repositorios externos. |

## Temas estudados

- Arrays NumPy.
- Operacoes vetorizadas.
- Leitura de dados numericos.
- Comparacao entre listas Python e arrays.
- Manipulacao e analise de bases numericas.
