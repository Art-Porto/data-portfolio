# Python para Data Science

Trilha de estudos da Alura voltada para fundamentos de Python, analise numerica e manipulacao de dados.

## Modulos

| Pasta | Tema | Conteudo principal |
| --- | --- | --- |
| `1-primeiros-passos/` | Primeiros passos | Variaveis, tipos, operadores, controle de fluxo e notebooks iniciais. |
| `2-funções-estruturas-de-dados-e-exceções/` | Funcoes e estruturas | Funcoes, listas, dicionarios, tuplas, excecoes e boas praticas. |
| `3-analise-numerica-NumPy/` | NumPy | Arrays, operacoes numericas, desempenho e bases de treino. |
| `4-pandas-conhecendo/` | Pandas inicial | DataFrames, Series, filtros e primeiros tratamentos. |
| `5-pandas-i-o/` | Entrada e saida | Leitura de CSV, XLSX, JSON, XML, HTML e outras fontes. |
| `6-pandas-selecionando-e-agrupando-dados/` | Selecao e agrupamento | Filtros, agrupamentos, agregacoes e desafios de analise. |

## Uso recomendado

Abra os notebooks na ordem dos modulos para acompanhar a progressao natural dos estudos. As bases de dados ficam dentro das proprias pastas dos modulos para manter cada exercicio reproduzivel.
