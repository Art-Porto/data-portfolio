# Alura

Pasta dedicada aos estudos feitos na Alura, com foco principal em Python para Data Science.

## Estrutura

| Pasta | Conteudo |
| --- | --- |
| `python-para-data-science/` | Trilha de Python, NumPy e Pandas, organizada por curso/modulo. |

## Como navegar

Comece por `python-para-data-science/README.md` para ver a ordem dos modulos. Cada subpasta possui notebooks, desafios e bases de dados usadas nas aulas.

## Principais temas

- Fundamentos de Python.
- Funcoes, estruturas de dados e tratamento de excecoes.
- Analise numerica com NumPy.
- Manipulacao de dados com Pandas.
- Entrada e saida de dados em diferentes formatos.
- Selecao, agrupamento e sumarizacao de dados.
